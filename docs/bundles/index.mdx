# Bundles

These are all of our Symfony bundles:


- **Bundle Helpers**
- **CLI**
- [**Embed Helpers**](./embed-helpers)<br />
  Helper to ease using external embeddable content (like from video platforms).
- [**Feature Flags**](./feature-flags)<br/>
  Bundle to integrate feature flags into Symfony.
- **Hosting**
- **HTML Builder**
- **Lenient Front Matter**
- **Menu**
- **PHP-CS**
- **PHP-CS-Fixer**
- **Prismic API**
- [**RAD**](./rad)<br />
  Bundle that adds often use features and default settings in Symfony.
- **Simple Normalizer**
- **Storyblok**
- **Task Manager**


## Deprecated Bundles

- **Assets**: use [Symfony Assets] instead.
- **Menu**: no replacement.
- **Umbrella**: use [Storybook] instead.
